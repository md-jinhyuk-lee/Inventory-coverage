# 📊 재고 커버리지 분석 대시보드

## 🎯 프로젝트 개요
회사 재고 관리를 위한 인터랙티브 웹 대시보드입니다. Excel 파일을 업로드하여 재고 커버리지를 실시간으로 분석하고 시각화할 수 있습니다.

## ✨ 주요 기능

### 📊 분석 메뉴
- **전체 요약**: 매장/상품/위험상품 현황
- **BIZ별 분석**: AP, FW, EQ 사업부별 상세 분석
- **시즌별 분석**: 시즌별 재고 현황 및 트렌드
- **매장별 상세 분석**: 매장별 성과 비교
- **상세 분석**: 필터링 및 정렬 기능
- **종합 리포트**: 경영진용 요약 리포트
- **이메일 발송**: 자동 리포트 이메일 발송

### 🔍 핵심 기능
- **재고 커버리지 자동 계산**: 현재재고량 ÷ 3주 평균 판매량
- **위험도 분류**: 🚨위험(2주미만), ⚠️주의(2-4주), ✅양호(4주이상)
- **인터랙티브 차트**: Plotly 기반 동적 시각화
- **Excel 다운로드**: 분석 결과 Excel 파일 다운로드
- **HTML 리포트**: 브라우저에서 볼 수 있는 리포트

## 📋 필수 데이터 형식

업로드할 Excel 파일에는 다음 컬럼이 포함되어야 합니다:

| 컬럼명 | 설명 | 예시 |
|--------|------|------|
| 매장명 | 매장 이름 | 강남점, 홍대점 |
| 상품명 | 상품 이름 | 나이키 에어맥스 |
| 상품코드 | 고유 상품 코드 | NK001, AD002 |
| BIZ | 사업부 구분 | AP, FW, EQ |
| 시즌 | 시즌 구분 | 24SS, 24FW |
| 소비자가 | 상품 가격 | 150000 |
| 1주차_판매량 | 1주차 판매 수량 | 10 |
| 2주차_판매량 | 2주차 판매 수량 | 8 |
| 3주차_판매량 | 3주차 판매 수량 | 12 |
| 현재_재고량 | 현재 재고 수량 | 50 |
| 재고_금액 | 재고 금액 | 2500000 |
